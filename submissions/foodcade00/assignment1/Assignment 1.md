# Introduction to Data Science and Machine Learning

## 1. Data Science vs Machine Learning
Data Science is the process of collecting, cleaning, analyzing, and interpreting data to extract insights for decision-making. Machine Learning is a subset of AI that allows systems to learn patterns from data and make predictions.

Relationship: Machine Learning is one of the main tools used in Data Science to build predictive models.

Example: In banking, Data Science analyzes transactions while Machine Learning detects fraud patterns.

---

## 2. Data Science Lifecycle
1. Problem Definition – Identify the problem.
2. Data Collection – Gather data from sources.
3. Data Cleaning – Fix missing or incorrect data.
4. Exploratory Data Analysis – Find patterns in data.
5. Feature Engineering – Select important variables.
6. Model Building – Apply Machine Learning algorithms.
7. Evaluation – Measure model performance.
8. Deployment – Use model in real systems.
9. Monitoring – Maintain and improve model.

ML is mainly used in model building and evaluation stages.

---

## 3. Supervised vs Unsupervised Learning

Supervised Learning:
- Uses labeled data
- Example: Email spam detection

Unsupervised Learning:
- Uses unlabeled data
- Example: Customer segmentation

---

## 4. Overfitting
Overfitting happens when a model learns training data too well and performs poorly on new data.

Causes:
- Complex model
- Small dataset
- Too many features

Prevention:
- More data
- Cross-validation
- Regularization
- Simpler models

---

## 5. Training vs Test Data
- Training data: Used to train the model
- Test data: Used to evaluate performance

Common split: 80% training, 20% testing

Why: To check how well the model works on unseen data and avoid overfitting.

---

## 6. Case Study: E-commerce Recommendation Systems
E-commerce platforms like Amazon use Machine Learning to recommend products based on user behavior.

Data used:
- Clicks
- Purchases
- Search history

Process:
- Data is collected and cleaned
- User behavior patterns are analyzed
- ML models predict what users may like

Impact:
- Better user experience
- Increased sales
- Personalized recommendations

Lifecycle coverage:
Covers all stages from problem definition to deployment.
