# README

Folder-kan waxaa ku jira dhammaan shaqooyinka aan ku gudbinayo DS & ML Bootcamp-ka, oo ay ku jiraan assignments-ka, layliyada, iyo Final Project-ka.

Ujeeddada folder-kan waa in aan hal meel ku ururiyo shaqadayda inta uu socdo bootcamp-ku, isla markaana aan si fudud ula socdo horumarkayga. Shaqooyin cusub ayaa lagu dari doonaa marka aan dhammeeyo qaybaha kala duwan ee barnaamijka.

Waxyaabaha laga heli karo folder-kan waxaa ka mid ah:

- Assignments-ka toddobaadlaha ah
- Layliyo iyo tababaro
- Faylasha mashruucyada
- Datasets iyo xogaha la isticmaalay
- Dukumentiyo iyo sharaxaadyo
- Final Project-ka iyo dhammaan waxyaabaha la xiriira

Fadlan ka taxaddar haddii aad wax ka beddelayso faylasha ku jira folder-kan. Kahor intaadan wax tirtirin, beddelin, ama rar-rarin, iska hubi inaad fahantay waxa uu faylkaasi yahay iyo saamaynta uu isbeddelku yeelan karo.

Folder-kan waa meesha aan ku kaydiyo uguna gudbiyo dhammaan shaqooyinka bootcamp-ka, sidaas darteed waxaa muhiim ah in qaab-dhismeedkiisa iyo faylashiisaba la ilaaliyo.
