

## 1. The definition of Data Science and Machine Learning
### Data Science 
Data Science is defined as a field of study that uses scientific methods, processes, and systems to extract knowledge and insights from data. It combines mathematics, statistics, and computer science to analyze and interpret data.
### Machine Learning 
Machine Learning is a subset of Artificial Intelligence that focuses on developing algorithms and statistical models that enable computer systems to improve their performance on specific tasks through experience.

## The relation ship b/w them

Relationship Between Data Science and Machine Learning
Data Science and Machine Learning are closely related because both work with data.

Data Science is a broad field that involves collecting, cleaning, analyzing, and interpreting data to gain insights. Machine Learning is a part of Artificial Intelligence that focuses on building models that learn from data and make predictions.

In simple terms, Data Science uses Machine Learning as a tool to find patterns and improve decision-making from data.

## 2. Data Science Lifecycle
	1. Obtaining Data  
	2. Cleaning Data  
	3. Exploring Data  
	4. Modeling Data  
	5. Interpreting Results  
###  Where machine learning fits in
Machine Learning mainly fits into the Modeling stage. At this stage, data has already been collected, cleaned, and explored. Machine Learning algorithms use this prepared data to learn patterns and make predictions.

 
## 3.Supervised Learning vs Unsupervised Learning

### Supervised learning

Supervised learning is when a model is trained using labeled data. Each input has a correct output.

Examples:
 - Credit card fraud detection  
 - Image classification  
 - Stock price prediction  

### Unsupervised learning

Unsupervised learning uses data without labels. The model finds patterns on its own.

Examples:
- Customer segmentation  
- Recommendation systems  
- Anomaly detection  

## 4. Overfitting
Overfitting happens when a model learns both patterns and noise in training data, causing poor performance on new data.

### Causes:
- Small dataset  
- Noisy data  
- Too complex model  
- Long training time  

### Prevention:
- Early stopping  
- Pruning  
- Regularization  
- Ensembling  
- Data augmentation  

## 5. Train-Test Split
### Data is split into:
- Training set (about 80%) → used to train the model  
- Test set (about 20%) → used to evaluate performance  

This is important to make sure the model works well on unseen data and does not overfit.

## 6. Role of Data Science in Business
Data Science helps organizations turn raw data into useful insights. It improves decision-making, increases efficiency, reduces risk, and helps businesses understand customers and markets.

### The part of the Data Science lifecycle it covers:
1. Define objectives  
2. Collect data  
3. Prepare data  
4. Analyze data  
5. Visualize results  
6. Generate insights  
7. Store and secure data
