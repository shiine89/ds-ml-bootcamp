# 1. Define Data Science and Machine Learning. What is the relationship between them?

Data Science is the process of collecting, cleaning, analyzing, and using data to solve problems and make decisions. Machine Learning is a part of Data Science that teaches computers to learn patterns from data and make predictions.

For example, in football, Data Science can analyze player statistics, while Machine Learning can predict which player is most likely to score in the next match.

# 2. Describe the Data Science lifecycle. At which stage does Machine Learning typically fit in, and why?

The Data Science lifecycle usually includes defining the problem, collecting data, cleaning data, analyzing data, building a model, evaluating it, and deploying the solution.

Machine Learning fits in the model-building stage because it is used to learn patterns from data and make predictions or decisions.

# 3. Compare Supervised Learning and Unsupervised Learning, giving an example of each.

Supervised Learning uses labeled data where the correct answer is already known. For example, predicting whether a football team will win or lose based on past match results.

Unsupervised Learning uses unlabeled data to find hidden patterns. For example, grouping football fans based on their ticket-buying behavior without knowing the groups beforehand.

# 4. What causes Overfitting? How can it be prevented?

Overfitting happens when a model memorizes the training data instead of learning general patterns, causing poor performance on new data.

It can be prevented by using more data, reducing model complexity, using cross-validation, and keeping separate data for testing.

# 5. Explain how training data and test data are split, and why this process is necessary.

A dataset is usually split into training data (for learning) and test data (for evaluation), such as 80% training and 20% testing.

This is necessary to check whether the model can perform well on new data rather than only on the data it has already seen.

# 6. Case Study: Machine Learning in Healthcare

One healthcare application of Machine Learning is detecting diseases such as sepsis from patient records. Researchers trained models using hospital data to identify patients at risk earlier than traditional methods.

This case study covers data collection, data cleaning, model building, and model evaluation stages of the Data Science lifecycle and helps doctors make faster decisions.
